# Aplikasi Undian Doorprize - PT GWI

Aplikasi undian doorprize untuk acara SEMARAK KEMERDEKAAN 80 TH

## Fitur
- Animasi acak nama pemenang
- Upload peserta via file Excel/CSV
- Histori pemenang
- Tombol start/stop
